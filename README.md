# 🏠 House Price Prediction using Machine Learning

## Project Overview

This project predicts the selling price of a house using Machine Learning. A Linear Regression model was trained on housing data to estimate house prices based on important features such as the number of bedrooms, bathrooms, living area, number of floors, and year built.

The project demonstrates the complete Machine Learning workflow, including data preprocessing, exploratory data analysis, model training, evaluation, and prediction.

## Features

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Selection
- Model Training using Linear Regression
- Model Evaluation
- House Price Prediction
- Model Saving using Joblib

## Dataset

The dataset contains information about residential houses, including:

- Bedrooms
- Bathrooms
- Living Area (sq ft)
- Floors
- Year Built
- House Price

Dataset Source:

https://www.kaggle.com/datasets/harlfoxem/housesalesprediction

## Machine Learning Workflow

1. Import Libraries
2. Load Dataset
3. Explore Dataset
4. Handle Missing Values
5. Select Features
6. Split Dataset into Training and Testing Sets
7. Train Linear Regression Model
8. Evaluate the Model
9. Save the Trained Model
10. Predict House Prices


## Model Evaluation

The model was evaluated using:

- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score

These metrics help measure the prediction accuracy of the model.
